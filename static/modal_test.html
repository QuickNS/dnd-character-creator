Modal Backdrop Test Suite

Test different approaches to having a visible backdrop that doesn't block clicks on the modal.

Approach 1: Default Bootstrap

Standard Bootstrap modal with backdrop: true

Expected: Works but might grey out content

Approach 2: Custom Z-Index

Modal with higher z-index than backdrop

Expected: Should layer above backdrop

Approach 3: Backdrop Pointer-Events None

Backdrop with pointer-events: none, dialog with pointer-events: auto

Expected: Clicks pass through backdrop

Approach 4: Fixed Position Dialog

Modal dialog with position: fixed and very high z-index

Expected: Force dialog on top

Approach 5: Manual Backdrop

Custom backdrop div with pointer-events: none, no Bootstrap backdrop

Expected: Full control over backdrop behavior

Approach 6: Transform Z-Space

Use CSS transform to put dialog in different z-space

Expected: 3D transform might help layering
